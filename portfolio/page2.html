Creative Work

As part of my Digital Technology and Culture coursework, I have taken several classes which have given me the opportunity to express my creativity. For example, in Multimedia Authoring I created a website for the poem "Thirteen Ways of Looking at a Blackbird" by Wallace Stevens. I used examples of my own photography to illustrate the poem and used html and CSS code to style the poem.

Recipe Project

One of my passions is cooking, creating and adapting recipes, so when I had the requirement to create a responsive design website of a recipe I was pretty excited. I later went back and added microdata to the website based on what I learned about schema.org through my Information Structures course. Feel free to try resizing the browser to see how the recipe layout changes at different widths of screens.
